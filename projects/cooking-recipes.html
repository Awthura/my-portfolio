Cooking Recipes Website - Team Leadership Project

Software Engineering | Web Development | Team Management

Project Overview

This is a University project for the Software Engineering module at THD (Technische Hochschule Deggendorf). I worked as a team leader and Agile Project Manager, leading a team of 11 students to develop a comprehensive web application that displays and filters cooking recipes from a database. The project showcased modern web development practices, team collaboration, and agile methodologies.

My Role

As the Software Agile Project Manager and Team Leader, I was responsible for coordinating the development efforts across frontend, backend, and database teams. I facilitated sprint planning, daily standups, and retrospectives while also contributing directly to the backend development. My leadership ensured that all team members stayed aligned with project goals and deadlines.

Technologies Used

Spring Boot Vue.js MongoDB Docker GitLab REST API

Key Features

  • Recipe browsing and filtering system
  • Database-driven content management
  • Responsive web interface built with Vue.js
  • RESTful API backend with Spring Boot
  • MongoDB for flexible data storage
  • Dockerized deployment for easy setup
  • Android application (in development)

Challenges & Learning

Leading a team of 11 students presented unique challenges in coordination and communication. I learned valuable lessons in agile project management, conflict resolution, and technical decision-making. The project also deepened my understanding of full-stack development and the importance of clear architecture in team environments.
