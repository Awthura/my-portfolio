Custom Object Detection with YOLO

Computer Vision | Transfer Learning | Thesis Project

Project Overview

My final year thesis project at Yangon Technological University (2019-2020). I customized the YOLO model to detect local Burmese traffic including pedestrians in traditional costumes, old buses, and unique vehicles like trishaws.

My Role

I handled the complete project including dataset collection, annotation, transfer learning implementation using Google Colab, and model evaluation for local traffic detection.

Technologies Used

YOLOPythonGoogle ColabTransfer LearningOpenCV

Key Features

  • Custom dataset for Burmese traffic
  • Transfer learning from pre-trained YOLO
  • Detection of unique local vehicles
  • Jupyter notebook implementation
