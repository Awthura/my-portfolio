Brain Tumor Classifier

Machine Learning | Medical AI | Classification

Project Overview

This project uses Machine Learning to classify brain tumors on a dataset from Kaggle. The UI provides interactive sliders to adjust features and displays the probability of a tumor based on the adjusted values.

My Role

I developed the complete machine learning pipeline including data preprocessing, model training, and creating an interactive UI for medical professionals to explore tumor classification predictions.

Technologies Used

PythonScikit-learnPandasStreamlitKaggle Dataset

Key Features

  • Interactive feature adjustment
  • Real-time probability calculation
  • Closest match image display
  • Statistical metrics
