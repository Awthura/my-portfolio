Machine Learning Model to Predict Chess ELO

Machine Learning | Research | Feature Engineering

Project Overview

This research project was completed as part of the Machine Learning module at THD (Technische Hochschule Deggendorf). Our team investigated how to create an ELO-guessing model based on a comprehensive chess games database. The goal was to predict player skill levels by analyzing game patterns, move sequences, and strategic decisions.

My Role

I worked as part of a research team, contributing to the development of feature extraction equations and the implementation of machine learning models. My responsibilities included data preprocessing, feature engineering, model training, and documenting our findings in a comprehensive research paper written in LaTeX.

Technologies Used

Python Scikit-learn Pandas NumPy LaTeX Jupyter Notebook

Key Features

  • Custom feature extraction from chess game data
  • Multiple ML algorithms tested and compared
  • Statistical analysis of chess patterns
  • ELO rating prediction based on game characteristics
  • Comprehensive research documentation
  • Performance metrics and model evaluation

Challenges & Learning

The main challenge was developing meaningful features from raw chess game data that could effectively capture player skill level. We experimented with various equations to extract patterns such as move quality, tactical accuracy, and strategic decision-making. This project deepened my understanding of feature engineering, the importance of domain knowledge in machine learning, and academic research methodology.
