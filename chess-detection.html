Chess Piece Detection with YOLO

Computer Vision | Object Detection | Deep Learning

Project Overview

This project was developed for the Computer Vision module at THD. We implemented YOLOv8 to localize and classify chess pieces on a chessboard. The system can detect individual pieces in real-time and display their classification alongside confidence levels.

My Role

I was responsible for implementing the YOLOv8 model, preparing the training dataset, fine-tuning the model for chess piece detection, and developing the real-time detection pipeline.

Technologies Used

Python YOLOv8 OpenCV PyTorch

Key Features

  • Real-time chess piece detection
  • Confidence level display
  • Webcam compatible
